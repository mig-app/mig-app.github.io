# mig-app.github.io

The public site for **mig** (миг): a speed-reading Russian vocabulary app for iOS.

Live at: https://mig-app.github.io

## Pages

- `index.html` : landing page
- `privacy.html`: privacy policy (App Store requirement)
- `support.html`: support contact and FAQs (App Store requirement)
- `style.css`: shared stylesheet, mig brand language

## About the app

mig drills Russian vocabulary at speed (50 to 1000 wpm) across paired, learning,
and Russian-only flash modes. Built for English speakers learning Russian.

Created by [Devin Dyson](https://github.com/ddyson1).
